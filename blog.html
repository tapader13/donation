What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a webpage as a hierarchical tree of nodes, where each node corresponds to an element or piece of content (like an HTML tag, attribute, or text). This allows scripts (like JavaScript) to access, modify, and interact with the HTML or XML document's content, structure, and style dynamically.

How do you select an element from the DOM?

To select an element from the DOM, JavaScript provides several methods that can be used based on different criteria, such as tag name, class, ID, or other attributes.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where a single event listener is attached to a parent element to manage events triggered by its child elements. Instead of attaching separate event listeners to each child element, event delegation takes advantage of the event bubbling phase, where events propagate upwards through the DOM tree.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes in the DOM, you can use `setAttribute()` to change an attribute, such as setting the `src` of an image. To adjust styles, you can modify an element's inline CSS properties via `element.style`, for example, `element.style.color = 'red'`. Additionally, you can use `classList.add()` or `classList.remove()` to dynamically apply or remove CSS classes from elements.

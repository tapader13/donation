What is the Document Object Model (DOM)?

It is the structure of a web page expressed as a Hierarchical tree of nodes, with each node representing an Element or fragment of content such as an HTML tag, attribute or text. This allows JavaScript, to access, navigate, transform, and change the content of an HTML document. structure, and style dynamically.

How do you select an element from the DOM?

To select an element from the DOM, JavaScript provides several methods that we can used based on different criteria, such as tag name, class name, ID name, or other attributes also we can use queryselector.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where a single event listener is attached to a parent element to manage events triggered by its child elements. Instead of attaching separate event listeners to each child element, event delegation takes advantage of the event bubbling phase, where events propagate upwards through the DOM tree.This way we can done our work by using one event listener.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate the attributes of an element we can use `setAttribute()` to set an attribute, also can use `removeAttribute()` for removing attribute from element. for example setting the `src` of an image. To change styles, we can replace an element's inline CSS properties via `element.style`, for example, `element.style.color = 'red'`. We can also dynamically add the classList.add() or classList.remove() CSS classes from elements.
